What is event delegation in the context of the DOM, and why is useful?

Event delegation is a technique in the DOM where you attach a single event listener to a parent element rather than adding multiple listeners to individual child elements. The event listener on the parent element takes advantage of event bubbling to catch events fired by its child elements. How Event Delegation Works: When an event (like a click) is triggered on an element, it bubbles up through the DOM tree, from the target element (where the event occurred) to its ancestors. In event delegation, you use this bubbling process to catch events on child elements from a single listener on a parent element. Example: Instead of adding a click event listener to each button in a list, you can attach one listener to the parent container and determine which button was clicked using the event.target property.
